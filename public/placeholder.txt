# Public Assets Directory

Add your images and videos to this directory:

## Directory Structure:

```
public/
├── videos/
│   └── hero-video.mp4        # Add your hero background video here
├── images/
│   ├── hero-poster.jpg       # Video poster/fallback image
│   ├── services/             # Service grid images
│   │   ├── lawn-care.jpg
│   │   ├── irrigation.jpg
│   │   ├── garden.jpg
│   │   └── hydroponics.jpg
│   └── before-after/         # Before/After transformation images
│       ├── before-1.jpg
│       └── after-1.jpg
```

## Tips:
- Optimize images for web (use WebP format when possible)
- Keep video file size under 10MB for faster loading
- Recommended image dimensions: 1920x1080 for hero, 800x800 for services
