# Logo Files Guide

## üìÅ Required Logo Versions

Place your logo files in this `public/logos/` directory with the following naming convention:

### **1. Full Logo (Horizontal) - REQUIRED**
- **File:** `logo-full.svg` (or `.png`)
- **Use:** Desktop navigation, footer
- **Size:** ~200-300px width, 60-80px height
- **Format:** SVG preferred (scalable), PNG fallback
- **Background:** Transparent

### **2. Full Logo - Light Version (White/Transparent)**
- **File:** `logo-full-light.svg` (or `.png`)
- **Use:** Navigation when over video/hero section (white text needed)
- **Size:** Same as full logo
- **Format:** SVG preferred
- **Background:** Transparent

### **3. Logo Icon/Icon Only (Square) - REQUIRED**
- **File:** `logo-icon.svg` (or `.png`)
- **Use:** Mobile navigation, favicon, social media
- **Size:** 512x512px (square)
- **Format:** SVG preferred
- **Background:** Transparent

### **4. Favicon**
- **File:** `favicon.ico`
- **Use:** Browser tab icon
- **Size:** 16x16, 32x32, 48x48 (multi-size .ico file)
- **Format:** .ico

## üé® Design Guidelines

### Full Logo (Horizontal)
- Should work on both light and dark backgrounds
- Include "GreenThumb KE" text
- Brand green (#80a536) for the "KE" part
- Black or dark gray for "GreenThumb" text
- Or use your actual logo design

### Logo Icon
- Should be recognizable even at small sizes (16x16px)
- Usually just the icon/symbol part
- Or a simplified version of full logo
- Must work on light and dark backgrounds

### Light Version
- White or very light colors
- For use on dark backgrounds (hero video, dark sections)
- Same design, just inverted colors

## üöÄ Quick Start

**Minimum setup:**
1. Add your main logo as `logo-full.png`
2. Add your icon as `logo-icon.png`
3. Add favicon as `favicon.ico`
4. The code will automatically use them!

**If you only have one logo file:**
- Rename it to `logo-full.png`
- The code will create a fallback for the icon
